# cloud-providers

Multi-cloud (AWS, Azure) abstraction layer.

## Usage

Abstraction for cloud provider APIs. See the [meta-repo](https://github.com/absolute-realms/financial-rails) for orchestration.

## Development

- Install dependencies: `npm install`
- Build: `npm run build`
- Test: `npm test`

Part of the [financial-rails](https://github.com/absolute-realms/financial-rails) monorepo.
