# shared-core

Protocols, question rendering, and shared utilities.

## Usage

Provides shared logic and utilities for all modules. See the [meta-repo](https://github.com/absolute-realms/financial-rails) for orchestration.

## Development

- Install dependencies: `npm install`
- Build: `npm run build`
- Test: `npm test`
